Exchange Rates from USD to Other Countries/Unions

China, Australia, the EU, Japan, Canada, Egypt and Russia

I selected an Exchange Rate API, which is "currency conversion on Autopilot." With this API, I am able to display the current exchange rates/currency rates of various countries from USD. I found this API on https://www.exchangerate-api.com/ . The options/parameters I used in implementing this page with this API were "base_rate" (set to USD as I want to show rates from USD to other currency), "conversion_rates" and the individual country codes (ex: "AUD" for Australian Dollar).

This application would be useful for a travel-agency website or an hotel website that is trying to promote tourism in foreign countries. This API would be useful to implement a page that is useful for clients who plan to travel abroad and need to plan their finances for upcoming trips. It would also be useful because some clients seek to travel abroad when currency/exchange rates are lower than usual. Another application would be for a banking/investment website that has clients all over the world and needs to cover international transactions. This API could be used in a banking app.
