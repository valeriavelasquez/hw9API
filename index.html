Exchange Rates for the US, Australia, the EU and Japan

This API has currency conversion on Autopilot and provides up to date currency exchange rate on various countries. The website I found it on is https://www.exchangerate-api.com. The options or parameters that I used in this API where the conversion rates and the currency code of the country I wanted to display. "conversion_rates" and the "[countrycode]". This API is useful in many, many contexts. A travel agency website could use this to help clients plan their finances for upcoming trips. It would also be useful for people investing money abroad or banking abroad.
